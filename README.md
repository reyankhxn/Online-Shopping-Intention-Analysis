# Online Shopping Intention Analysis

## Overview
This project predicts whether a customer will purchase a product based on their online shopping behavior.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Streamlit

## Machine Learning Model
Random Forest Classifier

## Accuracy
89.65%

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Purchase Prediction
- Confusion Matrix Visualization
- Streamlit Web Application

## Dataset
Online Shoppers Purchasing Intention Dataset
